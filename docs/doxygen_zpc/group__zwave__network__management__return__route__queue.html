Detailed Description

Small queue component for assigning return routes.

Small queue component that will cache and process the assign return route requests issued by the application.

◆ route_state_t

enum route_state_t
Enumerator
ROUTE_STATE_UNASSIGNED_ENTRY 

The entry is unassigned.

ROUTE_STATE_NOT_ESTABLISHED 

The entry is in use, but the return route has not been assigned yet.

ROUTE_STATE_IN_PROGRESS 

The entry is in use, and the return route is being established.

ROUTE_STATE_ESTABLISHED 

The entry is in use, the return route has been established and we wait a cooldown period before we re-use this entry.
